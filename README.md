# Smart Task Manager

A task management web application that helps users organize, prioritize, and track tasks efficiently.

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Priority system
- Task filtering
- Responsive design
- Local storage support

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Data Structures & Algorithms
